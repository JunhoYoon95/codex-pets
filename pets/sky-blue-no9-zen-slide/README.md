# Sky Blue No.9 Zen Slide

Unofficial custom Codex pet for personal, non-commercial use.

![Sky Blue No.9 Zen Slide preview](./preview.gif)

## Install

From the repository root:

```bash
mkdir -p ~/.codex/pets
cp -R pets/sky-blue-no9-zen-slide ~/.codex/pets/
```

Then restart Codex or open a new Codex window so the pet list refreshes.
